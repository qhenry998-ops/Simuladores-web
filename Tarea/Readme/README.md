## Descripción

En este desafio aplique estilos usando CSS, tamnbien ise que resuelva ejercicios matemáticos mediante JavaScript cada archivo separado por sus respectivas carpetas y losdos desafios estan en el mismo formulario 

### Simulador de Transferencia de Calor
En esta parte el codigo calcula la temperatura final de un objeto usando la Ley de Enfriamiento que es la siguente formula:

**Fórmula utilizada:**

T = Ts + (T0 - Ts) * e^(-k * t)

### Calculador de Combinaciones Complejas
En esta parte calcula el total de combinaciones posibles utilizando factoriales y combinaciones matemáticas, esta utilizando la siguiente formula 

**Fórmula utilizada:**

C(n, r) = n! / (r! * (n - r)!)
